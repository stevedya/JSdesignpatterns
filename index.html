Design Patterns

A easy guide to design patterns in JavaScript

What are patterns

Patterns are tried and tested reusable techniques which solve common programming problems. The patterns are rigorously tested and analyzed by developers to ensure the pattern is a trusted solution. It must meet at least three requirements: fitness of purpose, usefulness and applicability to be considered valid (Addy Osmani, 2017).

Design patterns in JavaScript

Design patterns are already used everyday in JavaScript. We use patterns in DOM selectors, even though document.getElementByClassname(“.item”) is more efficient in the browser, we still lean heavier towards the jQuery library where we can select the same item by writing $(“.item”) but sacrifice efficiency in the browser.
